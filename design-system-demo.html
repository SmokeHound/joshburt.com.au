Accessibility Features

Built-in A11y

  • Skip Link: Press Tab from top of page to reveal skip-to-content link
  • Main Landmark: All pages automatically get id="main"
  • Focus States: All interactive elements have visible focus indicators
  • ARIA Labels: Proper labeling on buttons, forms, and navigation
  • Keyboard Navigation: Full site navigable without mouse

Performance Features

Optimizations

  • Preload: Critical CSS preloaded for faster initial render
  • DNS Prefetch: Font CDN connections prefetched
  • Lazy Loading: Images load on demand
  • Error Tracking: Lightweight client-side monitoring

These optimizations are applied globally via shared-config.html
